# Faraday Rotation Analysis
Python-based analysis of Faraday Rotation experimental data and Verdet Constant determination.

## Overview
This project analyzes Faraday Rotation and optical polarization experimental data using Python.

## Objectives
- Verify Malus' Law
- Determine the Verdet Constant
- Perform regression analysis
- Visualize experimental results

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
